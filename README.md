# Magic Shine BI

Dashboard Streamlit para analizar sell out Autoplanet y activaciones comerciales.

## Sprint 1

Incluye:

- Carga de múltiples archivos Excel de sell out.
- Carga opcional de archivo de promotores.
- Normalización automática de la matriz diaria de ventas.
- KPIs ejecutivos.
- Filtros por mes, tienda, categoría, día y activación.
- Rankings de tiendas, productos y días.
- Lectura ejecutiva automática.
- Descarga de datos normalizados en CSV.

## Cómo publicar en Streamlit

- Repository: `MagicShine2026/magicshine-bi`
- Branch: `main`
- Main file path: `app.py`

## Seguridad de datos

No subir archivos Excel comerciales al repositorio. La app usa carga directa desde el navegador y procesa los datos en memoria.
